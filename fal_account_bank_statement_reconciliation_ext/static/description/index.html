Story behind the features

ODOO BEHAVIOR

Odoo proposes an advanced interface for bank statement importation and bank statement management but limit the check and the navigation on bank statement.

CLUEDOO BEHAVIOR

Cluedoo add extra features on bank statement :

  • Gap = Ending balance - Computed balance in order to easily check your gap and find it out.
  • Bank statement menu to review all your bank statement from a dedicated view.
  • Bank statement line menu to easily search a transaction.

    • MODULE +

    Improve navigation and analysis on bank statement and bank statement lines.

    CLUEDOO RECOMMENDATION

    To be installed on 100% of the database.

    Our Feature as Solution

    Improve Bank Statement Navigation with check and menus.

    How to Use this Module?

    Feature 1 :

    When you create a new bank statement, the computed field Gap = Ending balance - Computed balance is computed.

    Feature 2 :

    Bank statement menu is added on Accounting menu.

    Feature 3 :

    Bank Statements Journal Items Menu is added in Accounting Menu.

    Feature 4 :

    On Bank Statements, you can add a product and counterpart account. The bank statement line with the chosen product will not be reconciled.
